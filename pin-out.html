Pin-out del microcontrolador Espressif ESP32

El ESP32 tiene un total de 38 pines, de los cuales 34 son GPIO (entradas/salidas digitales), 18 son pines analógicos y 4 son pines especiales.
Pines de alimentación
Vext: Entrada de alimentación externa. Se puede utilizar para alimentar el ESP32 con una fuente de alimentación externa de 3,3 V o 5 V.
5V: Salida de alimentación de 5 V. Se puede utilizar para alimentar componentes externos que requieran 5 V.
3.3V: Salida de alimentación de 3,3 V. Se puede utilizar para alimentar componentes externos que requieran 3,3 V.
Pines de tierra
GND: Pin de tierra común. Se utiliza para conectar los componentes a tierra. Pines de GPIO
Los pines GPIO del ESP32 pueden utilizarse como entradas o salidas digitales. Pueden utilizarse para conectar sensores, actuadores y otros dispositivos externos.
GPIOs 0 a 31: Pines GPIO de propósito general. Pueden utilizarse como entradas o salidas digitales.
GPIOs 32 a 39: Pines GPIO especiales. Estos pines tienen funciones específicas, como la generación de PWM o la conexión de sensores capacitivos.
Pines analógicos
Los pines analógicos del ESP32 se pueden utilizar para leer señales analógicas, como las de sensores.
GPIOs 34 a 39: Pines analógicos. Pueden utilizarse para leer señales analógicas. Pines especiales
Los pines especiales del ESP32 tienen funciones específicas.
GPIO_12: Pin de strapping. Se utiliza para configurar algunas funciones del ESP32.
GPIO_0: Pin de reinicio. Se utiliza para reiniciar el ESP32.
GPIO_2: Pin de interrupción externa. Se utiliza para detectar eventos externos.
GPIO_15: Pin de salida de nivel alto. Se puede utilizar para generar un nivel alto en un pin GPIO.
GPIO_5: Pin de salida de nivel bajo. Se puede utilizar para generar un nivel bajo en un pin GPIO.
Descripción de los pines GPIO
Los pines GPIO del ESP32 se pueden dividir en dos grupos: pines GPIO de propósito general y pines GPIO especiales.
Pines GPIO de propósito general
Los pines GPIO de propósito general son los pines GPIO 0 a 31. Estos pines pueden utilizarse como entradas o salidas digitales.
Como entradas: Los pines GPIO de propósito general pueden utilizarse para leer señales digitales de sensores o de otros dispositivos externos.
Como salidas: Los pines GPIO de propósito general pueden utilizarse para controlar dispositivos externos, como actuadores o LED. Pines GPIO especiales
Los pines GPIO especiales son los pines GPIO 32 a 39. Estos pines tienen funciones específicas, como la generación de PWM o la conexión de sensores capacitivos.
GPIO_32 a 39: Pines GPIO analógicos. Pueden utilizarse para leer señales analógicas.
GPIO_32: Pin de salida de PWM. Se puede utilizar para generar una señal PWM en un pin GPIO.
GPIO_33: Pin de entrada de PWM. Se puede utilizar para leer una señal PWM de un pin GPIO.
GPIO_34: Pin de entrada para el sensor de temperatura. Se puede utilizar para leer la temperatura ambiente.
GPIO_35: Pin de entrada para el sensor de luz. Se puede utilizar para leer la intensidad de la luz.
GPIO_36: Pin de entrada para el sensor de humedad. Se puede utilizar para leer la humedad relativa del aire.
GPIO_37: Pin de entrada para el sensor de presión. Se puede utilizar para leer la presión atmosférica.
GPIO_38: Pin de entrada para el sensor de proximidad. Se puede utilizar para detectar la proximidad de un objeto.
GPIO_39: Pin de entrada para el sensor capacitivo. Se puede utilizar para detectar el toque de un objeto.
Cómo elegir los pines adecuados
Al elegir los pines adecuados para un proyecto, es importante considerar los siguientes factores:
La función del pin: ¿El pin se utilizará como entrada o como salida?
El tipo de señal: ¿La señal será digital o analógica?
La potencia: ¿El pin necesitará una fuente de alimentación externa?
Con estos factores en mente, se puede elegir los pines adecuados para el proyecto.
